📈 UpGainPulse v2.0

Multi-Ticker Headless ORB Bot for Paper Trading · Alpaca API · Melbourne Time

🤖 What It Does

UpGainPulse is a fully automated paper trading bot that runs overnight while you sleep (Melbourne time). It monitors 5 highly liquid US tickers during the regular trading session, identifies the Opening Range Breakout (ORB) pattern, and executes bracket orders with automatic take-profit and stop-loss. All trades go through Alpaca's paper trading API — no real money at risk.

🧮 Position Sizing Formula

Qty = (Risk ÷ Stop Loss $) ÷ Current Price × Multiplier
Qty = ($50 ÷ $0.10) ÷ $500 × 1.0 = 1 share (SPY @ $500)
# Example: SPY @ $500 Entry: $500.00 SL: $499.90 (-$0.10) TP: $500.20 (+$0.20, 2:1 RR) P&L if TP: +$0.20 P&L if SL: -$0.10

🏗️ Architecture

UpGainPulseEngine

  • Orchestrates all tickers
  • Spins up ORBStateMachine per ticker
  • Manages WebSocket subscription
  • Handles graceful shutdown

ORBStateMachine

  • Per-ticker state machine
  • Tracks ORB range [low, high]
  • Detects breakout condition
  • Daily automatic reset

AlpacaPaperTrader

  • Submits bracket orders
  • Validates buying power
  • Rate limiting (cooldown)
  • Error handling + retries

TradeLogger

  • SQLite + WAL mode
  • Thread-safe with locks
  • Logs every entry/exit
  • Powers analytics module
